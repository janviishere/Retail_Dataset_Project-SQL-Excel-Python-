# 🛒 Retail Dataset Analysis Project (SQL + Excel + Python)

## 📌 Project Overview

This project demonstrates an **end-to-end data analytics workflow** using **Excel, Python, and SQL** on a retail dataset. The goal is to transform raw data into **meaningful insights** that support business decision-making.

It covers the complete pipeline:
➡️ Data Cleaning → Data Processing → SQL Analysis → Insights Generation

---

## 🎯 Objectives

* Clean and preprocess raw retail data
* Perform exploratory data analysis (EDA)
* Extract business insights using SQL queries
* Build a strong foundation in **real-world data analytics workflow**

---

## 🛠️ Tools & Technologies

* **Python (Pandas, NumPy)** – Data cleaning & transformation
* **SQL** – Data querying & analysis
* **Excel** – Initial data exploration & preprocessing
* **Jupyter Notebook** – Analysis & visualization

## 🔄 Workflow

### 1️⃣ Data Cleaning (Excel + Python)

* Removed duplicates
* Handled missing values
* Standardized column formats
* Converted data types

### 2️⃣ Data Transformation (Python)

* Feature engineering
* Data normalization
* Structured dataset for SQL

### 3️⃣ SQL Analysis

* Aggregations (SUM, AVG, COUNT)
* Grouping & filtering
* Business queries like:

  * Top-selling products
  * Revenue by category
  * Customer behavior patterns

### 4️⃣ Insights Generation

* Identified high-performing products
* Analyzed sales trends
* Derived actionable business insights
